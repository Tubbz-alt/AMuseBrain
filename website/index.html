Muse Headgear

Muse: the brain sensing headband, is an electroencephalography (EEG) technology. EEG is a cell-established, non-invasive, harmless method of recording the electrical activity of groups of brain cells. EEG provides robust real-time insight into the brain; and Muse is the most versatile and easy-to-use EEG system available.

Arduino-UNO running from blinking eyes

The brain data is collected and then a user threshold is set prior for this task on initializing the interface. The interface recieves the binary value (0 or 1) from the python script. We then send the value over to the arduino over the serial/com port to control lights. The proof of concept has been shown by a LED Diode

To incoporate this feature into your interface please use the following package.

Arduino-UNO.pkg

The concept is thus shown in a very limited space and has an extremely vast scope for embedded programming, and home automation.
