Muse Headgear

Muse: the brain sensing headband, is an electroencephalography (EEG) technology. EEG is a cell-established, non-invasive, harmless method of recording the electrical activity of groups of brain cells. EEG provides robust real-time insight into the brain; and Muse is the most versatile and easy-to-use EEG system available.

The Attention Memory & Working Memory

Attention Memory

Extensive Cognitive Attentional Analysis could be carried using our interface system with Muse. Both long term and short-term attention analysis could be carried out over a given span of data recording. This could be particularly beneficial to students studying in school and working with children and people suffering from Attention Deficit Disorders and dyslexia

Working Memory

By using neural analysis decomposition techniques, we could extract working memory features to actually understand how much memory retention took place at a cognitive level. High correlation between working memory and literacy and arithmetic makes the analysis of memory super helpful to understand numerical math capabilities and logical understanding in people. Orthogonal Based n= 6 layer wavelet decomposition techniques in MATLAB and formulaic methods were used to come up with a numerical computation of attention and memory.

Arduino-UNO running from blinking eyes

The brain data is collected and then a user threshold is set prior for this task on initializing the interface. The interface recieves the binary value (0 or 1) from the python script. We then send the value over to the arduino over the serial/com port to control lights. The proof of concept has been shown by a LED Diode
